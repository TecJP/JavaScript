O JavaScript tem 3 formas de ser escrito em um site.
  • Primeira forma:

    • Antes da tag head ser finalizada, coloca uma tag script com fechamento, assim escrevendo dentro da mesma o código JavaScript.
    Se o JavaScript for inserido da forma citada acima, o JavaScript será carregado antes do seu html.

  • Segunda forma:

    • Usar uma tag script com fechamento no final do body e dentro desta tag, escrever seu código JavaScript.
    Se for inserido dessa forma, será carregado por último na página.

  • Terceira forma:

    • Dentro de uma tag script no final do body você chamar um arquivo externo.
    Se for inserido dessa forma, ele vai ficar buscando um arquivo externo de JavaScript para o site.
